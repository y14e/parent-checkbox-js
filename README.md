# parent-checkbox.ts

WAI-ARIA compliant [checkbox](https://www.w3.org/WAI/ARIA/apg/patterns/checkbox/) (mixed-state) pattern implementation in TypeScript.

## Usage

```ts
import ParentCheckbox from './parent-checkbox';

new ParentCheckbox(root);
// => ParentCheckbox
//
// root: HTMLElement
```
